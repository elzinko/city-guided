# Commands

Commandes Cursor pour iamthelaw : gestion des règles et de l'historique Git.

## Contenu

Ce répertoire contient les commandes pour :

- **Gérer les rulesets** : Créer, modifier, et activer des ensembles de règles
- **Git workflow** : Commits structurés et réorganisation de l'historique
- **Automatisation** : Régénération de `ENTRY.md` et application des conventions

## Utilisation

Chaque fichier `.md` dans `cursor/` est une commande accessible via Cursor. Les commandes guident l'agent avec des instructions détaillées sur le processus à suivre.
