# Modules

This directory contains modules (rulesets) from different sources.

## Directory Structure

| Directory | Description |
|-----------|-------------|
| `imports/` | Modules imported from external sources (file, URL, git) |
| `exports/` | Modules exported from this project (shareable) |
| `custom/` | Modules created locally by the user |

## Module Format

Each module is a directory containing a `ruleset.yaml` file.

See [../core/rulesets/README.md](../core/rulesets/README.md) for format details.
