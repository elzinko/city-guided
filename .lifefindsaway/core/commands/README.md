# Commands

Commandes Cursor pour lifefindsaway : idéation et gestion des features.

## Contenu

Ce répertoire contient les commandes pour :

- **Idéation** : Brainstorming et exploration d'idées
- **Structure** : Création d'epics et features avec templates
- **Organisation** : Gestion du planning et documentation des bugs
- **Phase A** : Exploration avec acceptation de l'incomplétude

## Utilisation

Chaque fichier `.md` dans `cursor/` est une commande accessible via Cursor. Les commandes suivent la méthode agile définie dans [../../README.md](../../README.md).
