# Core Modules

Built-in modules for lifefindsaway.

## Available Modules

| Module | Description |
|--------|-------------|
| `ideation-basics` | Basic guidelines for feature ideation |

## Module Format

```yaml
name: module-name
version: "1.0.0"
description: Module description
tags:
  - tag1
  - tag2
guidelines:
  - id: unique-id
    title: Guideline Title
    content: |
      Markdown content describing the guideline.
      Can be multiline.
```
